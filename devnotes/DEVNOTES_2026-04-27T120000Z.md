**摘要**

**背景與目的**

- 問題 1: 開發時出現錯誤訊息：

- 問題 2: 瀏覽器 runtime 拋出 SyntaxError：
  - "The requested module ... doesn't provide an export named: 'default'"，原因為不當使用 `import appEnv from '$app/environment'`（`$app/environment` 無 default export）。
- 目的：修正錯誤匯入、移除被廢棄的語法警告（`on:click` -> `onclick`），並確保專案在 `npm run check` 下無錯誤（警告可後續處理）。
