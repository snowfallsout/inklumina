# Colorfield

TypeScript rewrite of the MBTI particle installation.

## Run

```bash
npm install
npm start
```

## Dev

```bash
npm run dev
```

## Notes

- Open the display on the PC with `http://localhost:3000/display` so camera access works.
- Open the mobile join page on the phone with `http://<LAN_IP>:3000/join`.
- The display canvas includes pointer/touch repulsion, so finger interactions can push particles away.
